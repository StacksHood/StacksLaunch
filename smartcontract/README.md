# StacksLaunch Smart Contracts

This directory contains the Clarity smart contracts for the StacksLaunch fundraising platform.

## Contracts
- `campaign-manager.clar`: Manages the creation, tracking, and finalization of fundraising campaigns.

## Development
- **Framework**: Clarinet
- **Language**: Clarity 4

## Commands
```bash
# Check syntax
clarinet check

# Run tests
npm run test
```
